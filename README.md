# Netflix Homepage Clone

A responsive Netflix-inspired homepage and sign-in UI built with HTML and CSS.

## Project Structure

- `index.html`: Main homepage layout
- `signin.html`: Sign-in page
- `css/styles.css`: Styling for the project
- `images/`: Image assets used by the pages

## Features

- Hero section with background imagery
- Responsive sections and cards
- Sign-in page UI
- Clean folder structure for static frontend projects

## Tech Stack

- HTML5
- CSS3

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/ShreyaSangwa/netflix_homepage.git
   ```
2. Open the project folder.
3. Run `index.html` in your browser.
